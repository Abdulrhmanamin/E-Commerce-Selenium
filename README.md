# Saucedemo Login Plugin Automation
 
This project ensures the robustness and security of the Saucedemo login plugin. It employs the Page Object Model (POM) with Selenium WebDriver and TestNG for streamlined automation.
 
## Features
 
- **Login Functionality Testing:**
  - Automated verification under various scenarios.
  - Validation of successful login and error handling.
 
- **Page Object Model (POM):**
  - Enhanced script organization and maintenance.
  - Page classes for encapsulating page-specific elements.
 
- **Selenium WebDriver:**
  - Browser automation for different web browsers.
  - Implementation of key interactions.
 
- **TestNG Integration:**
  - Efficient test case management and execution.
  - Test suite configuration, parallel execution, and reporting.
 
- **Parameterization and Data-Driven Testing:**
  - Dynamic input data with parameterization.
  - Data-driven testing with various datasets.
 
- **Logging and Reporting:**
  - Integration of logging for event capture.
  - Detailed TestNG reports for comprehensive analysis.
 
## Benefits
 
- **Reliability:** Consistent and repeatable test executions.
- **Maintainability:** Easy updates and maintenance with the POM design.
 
## Getting Started
 
### Prerequisites
 
- Java Development Kit (JDK)
- Selenium WebDriver
- TestNG
 
### Installation
 
1. Clone the repository: `git clone https://github.com/Abdulrhmanamin/E-Commerce-Selenium.git`
2. Set up dependencies as per prerequisites.
 
### Usage
 
1. Configure the test environment in configuration files.
2. Run the automation script using TestNG.
